## System Requirements

Before installing ChatGPT to Webflow tools, ensure your system meets the minimum requirements for optimal performance.

## Platform Requirements

### VS Code Extension

<AccordionGroup>
  <Accordion title="Minimum Requirements">
    **Editor:** - Visual Studio Code 1.60.0 or higher **Operating System:** -
    Windows 10 (version 1903) or higher - macOS 10.15 (Catalina) or higher -
    Linux (Ubuntu 18.04+, CentOS 7+, RHEL 7+) **Hardware:** - 2GB RAM minimum -
    100MB available disk space - Internet connection for AI processing
  </Accordion>

  <Accordion title="Recommended Requirements">
    **Editor:** - Visual Studio Code 1.80.0 or higher **Operating System:** -
    Windows 11 or latest macOS/Linux **Hardware:** - 8GB RAM or higher - 200MB

    internet connection

</AccordionGroup>

## Troubleshooting Requirements

### Common Issues

<AccordionGroup>
  <Accordion title="Insufficient Memory">
    **Symptoms:** Slow performance, crashes, timeouts **Solutions:** - Close
    unnecessary applications - Increase virtual memory - Upgrade RAM if possible
  </Accordion>

  <Accordion title="Network Connectivity">
    **Symptoms:** Failed API calls, slow responses **Solutions:** - Check
    internet connection - Verify firewall settings - Test API endpoint
    accessibility
  </Accordion>

  <Accordion title="Browser Compatibility">
    **Symptoms:** Extension not working, UI issues **Solutions:** - Update
    browser to latest version - Clear browser cache and cookies - Disable
    conflicting extensions
  </Accordion>
</AccordionGroup>

## Checking Your System

### System Check Commands

<CodeGroup>

```bash check node version
node --version
npm --version
```

```bash check system info
# Windows
systeminfo

# macOS
system_profiler SPHardwareDataType

# Linux
lscpu && free -h
```

```bash check network
ping google.com
curl -I https://api.openai.com
```

</CodeGroup>

### Browser Compatibility Check

<Steps>
  <Step title="Check Browser Version">
    Navigate to `chrome://version/` (Chrome) or `about:` (Firefox) to see your
    browser version.
  </Step>
  <Step title="Test JavaScript">
    Open browser console and run `console.log('JavaScript works')`.
  </Step>
  <Step title="Check Extensions">
    Ensure no conflicting extensions are installed.
  </Step>
</Steps>
