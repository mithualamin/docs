## Overview

The Webflow conversion feature transforms your ChatGPT conversations into clean, semantic HTML and CSS code that's optimized for Webflow's design system. This guide covers all the conversion options and best practices.

## Conversion Methods

### VS Code Extension

<Steps>
  <Step title="Open Command Palette">
    Press `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (Mac).
  </Step>
  <Step title="Select Conversion Command">
    Type "HTFlow: Convert to Webflow" and select the command.
  </Step>
  <Step title="Paste ChatGPT Content">
    Paste your ChatGPT conversation or generated content.
  </Step>
  <Step title="Configure Options">
    Set conversion preferences and output format.
  </Step>
  <Step title="Generate Code">
    Click "Convert" to generate Webflow-ready code.
  </Step>
</Steps>

### Cursor Integration

<Steps>
  <Step title="Open AI Chat">Access the AI chat interface in Cursor.</Step>
  <Step title="Use Conversion Commands">
    Type conversion commands like `/convert-webflow` or `/webflow-html`.
  </Step>
  <Step title="Provide Content">
    Share your ChatGPT conversation or content to convert.
  </Step>
  <Step title="Review Output">
    Review and refine the generated Webflow code.
  </Step>
</Steps>

### Chrome Extension

<Steps>
  <Step title="Navigate to ChatGPT">
    Go to chat.openai.com and open your conversation.
  </Step>
  <Step title="Click Extension Icon">
    Click the HTFlow extension icon in your browser toolbar.
  </Step>
  <Step title="Select Content">
    Choose which parts of the conversation to convert.
  </Step>
  <Step title="Convert to Webflow">
    Click "Convert to Webflow" to generate the code.
  </Step>
  <Step title="Copy Code">
    Copy the generated HTML and CSS to your clipboard.
  </Step>
</Steps>

### Conversion Settings

<AccordionGroup>
  <Accordion title="Code Style">
    **Indentation:** 2 or 4 spaces, tabs **Line Length:** 80, 100, or 120
    characters **Formatting:** Prettier, ESLint compatible **Comments:** Include
    or exclude code comments
  </Accordion>

  <Accordion title="Webflow Integration">
    **Class Naming:** BEM, CSS Modules, or custom convention **Responsive:**
    Mobile-first or desktop-first approach **Components:** Break into reusable
    components **Assets:** Include placeholder images and icons
  </Accordion>

  <Accordion title="Optimization">
    **Minification:** Minify CSS and HTML output **Compression:** Optimize for
    file size **Performance:** Include performance optimizations
    **Accessibility:** Ensure WCAG compliance
  </Accordion>
</AccordionGroup>

## Troubleshooting

### Common Issues

<AccordionGroup>
  <Accordion title="Incomplete Conversion">
    **Cause:** Ambiguous or incomplete ChatGPT prompts **Solution:** Provide
    more specific, detailed prompts with clear structure requirements
  </Accordion>

  <Accordion title="Styling Issues">
    **Cause:** Missing design specifications in prompts **Solution:** Include
    specific colors, fonts, spacing, and layout requirements
  </Accordion>

  <Accordion title="Responsive Problems">
    **Cause:** No responsive design instructions **Solution:** Explicitly
    request mobile-first or responsive design in prompts
  </Accordion>
</AccordionGroup>
