## How ChatGPT to Webflow Works

Follow this simple workflow to convert your ChatGPT conversations to Webflow:

### Step 1: Access HTFlow Custom GPT

<Card title="1. Access HTFlow Custom GPT" icon="brain">
  Visit our Custom GPT directly in ChatGPT for the most streamlined experience.

### Step 2: Generate Components

<Card title="2. Generate Components" icon="code">
    Use the Custom GPT to generate your HTML, CSS, and JavaScript components.

    **What the Custom GPT Does:**

    - Generates Webflow-compatible HTML structure
    - Creates responsive CSS with Webflow best practices
    - Provides JavaScript for interactive elements
    - Optimizes code for Webflow Designer import

    **Example Prompts:**

    - "Create a modern hero section with gradient background"
    - "Build a pricing table with 3 tiers"
    - "Design a contact form with validation"
    - "Make a testimonial slider component"

</Card>

### Step 3: Convert with HTFlow Web App

<Card title="3. Convert with HTFlow Web App" icon="webflow">
    Use the HTFlow web app to convert your generated code into Webflow components.

    **Conversion Process:**

    1. Copy the HTML and CSS from the Custom GPT
    2. Visit [HTFlow Webflow App](https://webflow.com/apps/detail/htflow)
    3. Paste your code into the input areas
    4. Click "Convert & Copy" to get Webflow-ready code
    5. Paste directly into Webflow Designer

    <Card title="HTFlow Web App" icon="external-link" href="https://webflow.com/apps/detail/htflow">
      Convert HTML/CSS to Webflow components
    </Card>

</Card>

### Step 4: Import to Webflow

<Card title="4. Import to Webflow" icon="upload">
    Paste the converted code directly into your Webflow project.

    **Import Steps:**

    - **Mac**: `Cmd + V` to paste into Webflow Designer
    - **Windows**: `Ctrl + V` to paste into Webflow Designer
    - Customize styling using Webflow's visual editor
    - Add interactions and animations as needed

</Card>
