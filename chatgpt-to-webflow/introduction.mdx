## How ChatGPT to Webflow Works

Follow this simple 4-step workflow to convert your ChatGPT conversations to Webflow:

<CardGroup>
  <Card title="Step 1: Access Custom GPT" icon="brain">
    Visit our HTFlow Custom GPT directly in ChatGPT for the most streamlined experience.
    
    **What You Get:**
    - Pre-trained for Webflow compatibility
    - Instant HTML, CSS, and JS generation
    - No setup or configuration required
    
    <Card title="Start Now" icon="external-link" href="https://chatgpt.com/g/g-68f0dcfaca7c81918ff288634d24b792-htflow-html-to-webflow">
      Open HTFlow Custom GPT
    </Card>
  </Card>

  <Card title="Step 2: Generate Components" icon="code">
    Use simple prompts to generate Webflow-compatible components instantly.

    **Example Prompts:**
    - "Create a modern hero section with gradient background"
    - "Build a pricing table with 3 tiers"
    - "Design a contact form with validation"
    - "Make a testimonial slider component"

    **What You Get:**
    - Clean, semantic HTML structure
    - Responsive CSS with Webflow best practices
    - Interactive JavaScript elements
    - Optimized for Webflow Designer import

  </Card>

  <Card title="Step 3: Convert with HTFlow App" icon="webflow">
    Use the HTFlow Webflow app to convert your generated code into native Webflow components.

    **Conversion Process:**
    1. Copy HTML and CSS from Custom GPT
    2. Open HTFlow app in Webflow Designer
    3. Paste your code into the input areas
    4. Click "Convert & Copy" to get Webflow-ready code
    5. Paste directly into your Webflow project

    <Card title="Install App" icon="external-link" href="https://webflow.com/apps/detail/htflow">
      Get HTFlow Webflow App
    </Card>

  </Card>

  <Card title="Step 4: Import to Webflow" icon="upload">
    Paste the converted code directly into your Webflow project and customize as needed.

    **Import Steps:**
    - **Mac**: `Cmd + V` to paste into Webflow Designer
    - **Windows**: `Ctrl + V` to paste into Webflow Designer
    - Customize styling using Webflow's visual editor
    - Add interactions and animations as needed

    **Result:**
    - Fully responsive components
    - Editable Webflow elements
    - Clean, optimized code
    - Ready for production

  </Card>
</CardGroup>
