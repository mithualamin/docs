## Overview

HTFlow streamlines the process of converting AI-generated HTML/CSS/JavaScript into production-ready Webflow sites. This guide walks you through the complete workflow.

  <img src="/images/hero-light.png" alt="HTFlow Workflow" />
</Frame>

## The HTFlow Process

HTFlow follows a simple three-step workflow:

<Steps>
  <Step title="Set Up">
    Configure your development environment with HTFlow tools
  </Step>
  
  <Step title="Build">
    Create your website using AI tools with HTFlow standards
  </Step>
  
  <Step title="Deploy">
    Convert and deploy your code to Webflow
  </Step>
</Steps>

## Step 2: Build with AI

<Tabs>
  <Tab title="Setup HTML Project">
    ### Initialize HTFlow Project
    
    <Steps>
      <Step title="Create Project Folder">
        ```bash
        mkdir my-webflow-project
        cd my-webflow-project
        code .  # Open in VS Code
        ```
      </Step>
      
      <Step title="Initialize HTFlow">
        Open HTFlow Panel (`Ctrl+Shift+H`) and click **"Initialize Project"**
        
        Or via command line:
        ```bash
        htflow init
        ```
      </Step>
      
      <Step title="Choose Template">
        Select a template to start with:
        - **Basic**: Minimal setup
        - **Landing Page**: Single page template
        - **Portfolio**: Multi-page template
        - **E-commerce**: Product site structure
      </Step>
    </Steps>
    
    ### Project Structure
    
    ```text
    my-webflow-project/
    ├── index.html
    ├── styles/
    │   └── main.css
    ├── scripts/
    │   └── main.js
    ├── assets/
    │   ├── images/
    │   └── fonts/
    └── htflow.config.js
    ```
  </Tab>
  
  <Tab title="Create with AI">
    ### Use AI to Build Your Site
    
    <AccordionGroup>
      <Accordion title="With Cursor AI" icon="sparkles">

  <Tab title="Refine & Test">
    ### Development Server
    
    <Steps>
      <Step title="Start Dev Server">
        ```bash
        # Via extension
        HTFlow Panel → "Start Development Server"
        
        # Or command line
        htflow dev
        ```
      </Step>
      
      <Step title="Preview in Browser">
        Server runs at `http://localhost:3000`
        
        <Check>Hot reloading enabled - changes appear instantly</Check>
      </Step>
      
      <Step title="Test Responsiveness">
        Use browser DevTools to test different screen sizes:
        - Desktop (1920px+)
        - Tablet (768px - 1024px)
        - Mobile (320px - 767px)
      </Step>
    </Steps>
    
    ### Refine with AI
    
    Continue iterating with your AI assistant:
    
    ```text Example Iterations
    "Make the hero section taller"
    "Change the color scheme to blue and white"
    "Add hover effects to the buttons"
    "Improve mobile spacing"
    "Add smooth scroll animations"
    ```
    
    ### Validate Continuously
    
    ```bash
    # Run validation
    htflow validate
    
    # Fix any issues reported
    # Re-validate until clean
    ```
  </Tab>
</Tabs>

## Step 3: Deploy to Webflow

<Tabs>
  <Tab title="Prepare for Conversion">
    ### Run Final Checks
    
    <Steps>
      <Step title="Validate Code">
        ```bash
        htflow validate
        ```
        
        Ensure all checks pass:
        - ✓ HTML validation
        - ✓ CSS validation
        - ✓ JavaScript validation
        - ✓ Webflow compatibility
      </Step>
      
      <Step title="Run Project Audit">
        ```bash
        htflow audit
        ```
        
        Review scores:
        - Performance: 85+
        - Accessibility: 90+
        - Best Practices: 90+
        - SEO: 85+
        - Webflow Compatibility: 100
      </Step>
      
      <Step title="Build Production Code">
        ```bash
        # Via extension
        HTFlow Panel → "Build Project"
        
        # Or command line
        htflow build
        ```
        
        Output created in `./dist` folder
      </Step>
    </Steps>
  </Tab>
  
  <Tab title="Convert with Webflow App">
    ### Using HTFlow Webflow App
    
    <Steps>
      <Step title="Open Webflow">
        Log into your Webflow account at [webflow.com](https://webflow.com/)
      </Step>
      
      <Step title="Create New Site">
        - Click "Create New Site"
        - Choose blank template
        - Name your site
      </Step>
      
      <Step title="Install HTFlow App">
        - Go to Webflow Apps
        - Search for "HTFlow"
        - Install HTFlow app
      </Step>
      
      <Step title="Import Your Code">
        - Open HTFlow app in Webflow
        - Upload your `dist` folder
        - Or connect to your repository
        
        <Info>HTFlow will automatically convert your HTML/CSS/JS</Info>
      </Step>
      
      <Step title="Review & Customize">
        - Review converted design
        - Make adjustments in Webflow
        - Add CMS if needed
      </Step>
      
      <Step title="Publish">
        Click "Publish" to make your site live!
        
        <Check>Your AI-generated site is now live on Webflow</Check>
      </Step>
    </Steps>
  </Tab>
  
  <Tab title="Convert with Chrome Extension">
    ### Using HTFlow Chrome Extension
    
    <Steps>
      <Step title="Install Extension">
        - Open Chrome Web Store
        - Search "HTFlow"
        - Install HTFlow Chrome Extension
      </Step>
      
      <Step title="Open Your Build">
        Open your production build in Chrome:
        ```bash
        # Serve the dist folder
        cd dist
        python3 -m http.server 8000
        # Or use any static server
        ```
        
        Navigate to `http://localhost:8000`
      </Step>
      
      <Step title="Activate Extension">
        - Click HTFlow extension icon
        - Click "Convert to Webflow"
      </Step>
      
      <Step title="Review Conversion">
        Extension analyzes your page and prepares conversion:
        - Scans HTML structure
        - Extracts CSS classes
        - Identifies interactions
      </Step>
      
      <Step title="Export to Webflow">
        - Click "Export"
        - Choose export method:
          - Direct to Webflow account
          - Download conversion package
        
        <Check>Code is ready for Webflow import</Check>
      </Step>
      
      <Step title="Import in Webflow">
        - Open your Webflow site
        - Import the conversion package
        - Review and publish
      </Step>
    </Steps>
  </Tab>
</Tabs>

## Tips for Success

<Tabs>
  <Tab title="AI Prompting">
    ### Effective AI Prompts
    
    **Be Specific:**
    ```text ❌ Bad
    "Create a website"
    
    ✅ Good
    "Create a landing page with:
    - Hero section with gradient background
    - 3-column feature section with icons
    - CTA section with form
    - Mobile responsive
    - Webflow-compatible classes"
    ```
    
    **Iterate Gradually:**
    ```text
    1. "Create basic structure"
    2. "Add styling to hero section"
    3. "Make navigation sticky"
    4. "Add hover animations"
    ```
    
    **Request Webflow Compatibility:**
    ```text
    "Follow Webflow best practices:
    - Use semantic HTML
    - BEM-style class naming
    - Mobile-first CSS
    - No external dependencies"
    ```
  </Tab>
  
  <Tab title="Code Quality">
    ### Maintain Clean Code
    
    **Organize Files:**
    ```text
    styles/
    ├── base.css          # Reset, variables
    ├── layout.css        # Grid, containers
    ├── components.css    # Buttons, cards
    └── utilities.css     # Helper classes
    ```
    
    **Use CSS Variables:**
    ```css
    :root {
      --color-primary: #16A34A;
      --color-secondary: #07C983;
      --spacing-unit: 1rem;
      --font-heading: 'Inter', sans-serif;
    }
    ```
    
    **Comment Complex Sections:**
    ```html
    <!-- Hero Section: Full-height gradient background -->
    <section class="hero">
      <!-- Content -->
    </section>
    ```
  </Tab>
  
  <Tab title="Performance">
    ### Optimize for Speed
    
    **Image Optimization:**
    ```text
    - Use WebP format
    - Compress images (80-85% quality)
    - Lazy load below-fold images
    - Use appropriate dimensions
    ```
    
    **CSS Optimization:**
    ```text
    - Remove unused styles
    - Combine media queries
    - Minimize CSS specificity
    - Use shorthand properties
    ```
    
    **JavaScript Optimization:**
    ```text
    - Minimize DOM manipulation
    - Use event delegation
    - Defer non-critical scripts
    - Keep bundles small
    ```
  </Tab>
</Tabs>
