## Overview

The HTFlow VS Code extension provides a visual interface for all HTFlow CLI commands, making it easy to manage HTFlow projects without touching the terminal.

<Info>Works with both **Visual Studio Code** and compatible forks</Info>

## Prerequisites

Before installing the HTFlow extension:

<CardGroup cols={2}>
  <Card title="VS Code 1.99.0+" icon="window">
    Download from [code.visualstudio.com](https://code.visualstudio.com/)
  </Card>
  <Card title="Node.js 16+" icon="node-js">
    Required for HTFlow CLI Download from [nodejs.org](https://nodejs.org/)
  </Card>
</CardGroup>

## Installation

<Steps>
  <Step title="Open VS Code">Launch Visual Studio Code on your computer</Step>

  <Step title="Open Extensions Panel">
    Press `Ctrl+Shift+X` (Windows/Linux) or `Cmd+Shift+X` (Mac)
    <Tip>You can also click the Extensions icon in the Activity Bar</Tip>
  </Step>

  <Step title="Search for HTFlow">
    In the search bar, type **"Htflow Html to Webflow"**
    <Frame>
      <iframe
        width="560"
        height="315"
        src="https://www.youtube.com/embed/MBBAj5EaKSg"
        title="HTFlow Extension Tutorial"
        frameborder="0"
        allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share"
        allowfullscreen
      ></iframe>
    </Frame>
  </Step>

  <Step title="Install Extension">
    Click the **Install** button next to the HTFlow extension
  </Step>

  <Step title="Reload VS Code">
    Reload VS Code to activate the extension
    <Check>HTFlow icon should appear in the Activity Bar</Check>
  </Step>
</Steps>

## Install HTFlow CLI

After installing the extension, install the HTFlow CLI:

<Tabs>
  <Tab title="Via Extension (Recommended)">
    <Steps>
      <Step title="Open HTFlow Panel">
        Click the HTFlow icon in Activity Bar or press `Ctrl+Shift+H` /
        `Cmd+Shift+H`
        <Frame>
          <img
            src="/images/image__htflow-extension-open-vs.png"
            alt="HTFlow Extension Panel"
          />
        </Frame>
      </Step>

      <Step title="Click Install CLI">
        In the HTFlow panel, click **"Install HTFlow CLI"** button
      </Step>

      <Step title="Wait for Installation">
        Monitor progress in the Output panel (typically 1-2 minutes)
        <Check>Success message appears when complete</Check>
      </Step>
    </Steps>
  </Tab>

  <Tab title="Via Terminal">
    ```bash npm install -g htflow-cli ``` Verify installation: ```bash htflow
    version ```
  </Tab>
</Tabs>

## Extension Settings

Customize the extension behavior:

<Tabs>
  <Tab title="Access Settings">
    1. Open VS Code Settings (`Ctrl+,` / `Cmd+,`)
    2. Search for "HTFlow"
    3. Configure options
  </Tab>
  
  <Tab title="Available Settings">
    ```json settings.json
    {
      // Extension Settings
      "htflow.autoInstallCLI": true,
      "htflow.showWelcome": true,
      "htflow.checkUpdates": true,
      
      // Panel Settings
      "htflow.panel.position": "right",
      "htflow.panel.showStatusBar": true,
      
      // Development Server
      "htflow.dev.autoStart": false,
      "htflow.dev.defaultPort": 3000,
      "htflow.dev.openBrowser": true,
      
      // Validation
      "htflow.validation.validateOnSave": true,
      "htflow.validation.showInlineErrors": true,
      
      // Build
      "htflow.build.showNotifications": true,
      "htflow.build.autoClean": false,
      
      // Terminal
      "htflow.terminal.showOutput": true,
      "htflow.terminal.clearBeforeCommand": false
    }
    ```
  </Tab>
</Tabs>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Extension Not Appearing" icon="triangle-exclamation">
    **Solutions:** 1. Reload VS Code window (`Ctrl+R` / `Cmd+R`) 2. Check
    Extensions panel - ensure it's enabled 3. Verify VS Code version (needs
    1.99.0+) 4. Reinstall extension
  </Accordion>

  <Accordion title="Commands Not Working" icon="circle-xmark">
    **Check:** 1. HTFlow CLI is installed: `htflow version` 2. Node.js is
    installed: `node --version` 3. Check Output panel for errors 4. Restart VS
    Code
  </Accordion>

  <Accordion title="Panel Won't Open" icon="window">
    **Try:** 1. Use Command Palette: `HTFlow: Open Panel` 2. Check keyboard
    shortcut conflicts 3. Reset panel position in settings 4. Reload window
  </Accordion>

  <Accordion title="Server Issues" icon="server">
    **Solutions:** 1. Check port availability 2. Stop other servers on same port
    3. Change default port in settings 4. Check firewall settings
  </Accordion>
</AccordionGroup>
