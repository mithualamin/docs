## Overview

The HTFlow VS Code extension provides a visual interface for all HTFlow CLI commands, making it easy to manage HTFlow projects without touching the terminal.

<Info>Works with both **Visual Studio Code** and compatible forks</Info>

## Prerequisites

Before installing the HTFlow extension:

<CardGroup cols={2}>
  <Card title="VS Code 1.99.0+" icon="window">
    Download from [code.visualstudio.com](https://code.visualstudio.com/)
  </Card>
  <Card title="Node.js 16+" icon="node-js">
    Required for HTFlow CLI Download from [nodejs.org](https://nodejs.org/)
  </Card>
</CardGroup>

## Installation

<Steps>
  <Step title="Open VS Code">
    Launch Visual Studio Code on your computer
  </Step>
  
  <Step title="Open Extensions Panel">
    Press `Ctrl+Shift+X` (Windows/Linux) or `Cmd+Shift+X` (Mac)
    
    <Tip>You can also click the Extensions icon in the Activity Bar</Tip>
  </Step>
  
  <Step title="Search for HTFlow">
    In the search bar, type **"Htflow Html to Webflow"**
    
    <Frame>
      <iframe
        width="560"
        height="315"
        src="https://www.youtube.com/embed/MBBAj5EaKSg"
        title="HTFlow Extension Tutorial"
        frameborder="0"
        allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share"
        allowfullscreen
      ></iframe>
    </Frame>
  </Step>
  
  <Step title="Install Extension">
    Click the **Install** button next to the HTFlow extension
  </Step>

  <Step title="Reload VS Code">
    Reload VS Code to activate the extension
    <Check>HTFlow icon should appear in the Activity Bar</Check>
  </Step>
</Steps>

## Install HTFlow CLI

After installing the extension, install the HTFlow CLI:

<Tabs>
  <Tab title="Via Extension (Recommended)">
    <Steps>
      <Step title="Open HTFlow Panel">
        Click the HTFlow icon in Activity Bar or press `Ctrl+Shift+H` /
        `Cmd+Shift+H`
        <Frame>
          <img
            src="/images/image__htflow-extension-open-vs.png"
            alt="HTFlow Extension Panel"
          />
        </Frame>
      </Step>

      <Step title="Click Install CLI">
        In the HTFlow panel, click **"Install HTFlow CLI"** button
        <Frame>
          <img
            src="/images/image__htflow-extension-open-vs.png"
            alt="HTFlow Extension Panel"
          />
        </Frame>
      </Step>

      <Step title="Initialize Project">
        In the HTFlow panel, click **"Install HTFlow CLI"** button
        <Frame>
          <img
            src="/images/image__htflow-extension-vs-initialize-project.png"
            alt="HTFlow Extension Panel"
          />
        </Frame>
      </Step>

      <Step title="Audit and Validate">
        In the HTFlow panel, click **"Server"** and then click the **"Audit Project"** button to validate your code against HTFlow rules.

        Before running the audit, you need to create your HTML webpage using an AI agent. We've shown the default Copilot Agent for VS Code in this example,
        but you can use any AI agent you prefer, such as Cursor, Claude Code, GPT Code, Gemini CLI, or any other AI coding assistant.

        The key requirement is to instruct your AI agent to follow the HTFlow rules that were established during the project initialization step.
        You can provide the `.cursor` folder, `htflowrc.json`, and `AI_prompt.md` files as context to your AI agent (Copilot or others).
        This ensures the AI agent will generate HTML, CSS, and JavaScript code that is compatible with Webflow and Droip conversion requirements.

        Once you're satisfied with your design, you can run the audit from the HTFlow VS Code extension panel.
        Click on the **"Server"** tab and then click the **"Audit Project"** button. HTFlow will execute the `npx htflow audit` command in the terminal.

        The audit will display all rule violations that the AI might have introduced. If you encounter any errors, simply copy the entire terminal output
        and paste it into your AI agent's chat box, asking the AI to fix the issues. The AI will resolve all the errors for you.
        You can run the audit repeatedly until you achieve zero errors. Once you have error-free code, it's ready for Webflow or Droip conversion.
        <Frame>
          <img
            src="/images/image__htflow-extension-vs-audit.png"
            alt="HTFlow Extension Panel"
          />
        </Frame>

        <Info>
          See the <a href="/htflow/tutorial" target="_blank">complete tutorial</a> to learn how to build a full website with HTFlow from start to finish.
        </Info>
      </Step>

      <Step title="Start Dev Server">
        In the HTFlow panel, click **"Install HTFlow CLI"** button
