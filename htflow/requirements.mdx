## Software Dependencies

### Required

<Tabs>
  <Tab title="Node.js">
    **Why needed:** HTFlow CLI runs on Node.js
    
    **Installation:**
    
    <CodeGroup>
    ```bash macOS (Homebrew)
    brew install node
    ```
    

    ```bash Linux (apt)

    sudo apt-get install -y nodejs
    ```
    </CodeGroup>
    
    **Verify installation:**
    ```bash
    node --version
    # Should output: v18.x.x or higher
    ```
  </Tab>
  
  <Tab title="npm/Package Manager">
    **Why needed:** Install and manage HTFlow CLI and dependencies
    
    **Comes with Node.js**, or install alternatives:
    
    ```bash Yarn
    npm install -g yarn
    ```
    
    ```bash pnpm
    npm install -g pnpm
    ```
    
    **Verify installation:**
    ```bash
    npm --version
    # Should output: 8.x.x or higher
    ```
  </Tab>
  
  <Tab title="VS Code / Cursor">
    **Why needed:** IDE for development and HTFlow extension
    
    **Options:**
    - **VS Code**: [code.visualstudio.com](https://code.visualstudio.com/)


## Firewall Configuration

For development server access:

<Tabs>
  <Tab title="macOS">
    ```bash Allow Node.js
    # System will prompt automatically
    # Or manually add in System Preferences:
    # Security & Privacy → Firewall → Firewall Options
    # Allow incoming connections for Node.js
    ```
  </Tab>
  
  <Tab title="Windows">
    ```powershell Windows Firewall
    # Add firewall rule for Node.js
    New-NetFirewallRule -DisplayName "HTFlow Dev Server" `
      -Direction Inbound `
      -Program "C:\Program Files\nodejs\node.exe" `
      -Action Allow
    ```
  </Tab>
  
  <Tab title="Linux">
    ```bash ufw
    # Allow Node.js through firewall
    sudo ufw allow from 192.168.1.0/24 to any port 3000
    
    # Or allow all local connections
    sudo ufw allow in on lo
    ```
  </Tab>
</Tabs>

## Troubleshooting Setup

<AccordionGroup>
  <Accordion title="Node.js Not Found" icon="circle-xmark">
    **Issue:** `node` command not recognized
    
    **Solutions:**
    1. Reinstall Node.js
    2. Add Node.js to PATH:
       - Windows: Edit System Environment Variables
       - macOS/Linux: Add to `.bashrc` or `.zshrc`
    3. Restart terminal/VS Code
  </Accordion>
  
  <Accordion title="Permission Denied" icon="lock">
    **Issue:** Cannot install global packages
    
    **Solutions:**
    ```bash macOS/Linux
    # Use sudo (not recommended for regular use)
    sudo npm install -g htflow-cli
    
    # Or fix npm permissions (recommended)
    mkdir ~/.npm-global
    npm config set prefix '~/.npm-global'
    echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
    ```
    
    ```powershell Windows
    # Run as Administrator
    # Right-click terminal → Run as Administrator
    npm install -g htflow-cli
    ```
  </Accordion>
  
  <Accordion title="Extension Not Working" icon="puzzle-piece">
    **Issue:** HTFlow extension doesn't appear or work
    
    **Solutions:**
    1. Verify VS Code version: `code --version`
    2. Reload window: `Ctrl+Shift+P` → "Reload Window"
    3. Check extension is enabled in Extensions panel
    4. Reinstall extension
    5. Check for conflicting extensions
  </Accordion>
  
  <Accordion title="Port Already in Use" icon="network-wired">
    **Issue:** Cannot start dev server
    
    **Solutions:**
    ```bash
    # Find process using port 3000
    # macOS/Linux
    lsof -i :3000
    
    # Windows
    netstat -ano | findstr :3000
    
    # Kill the process or change port in config
    ```
  </Accordion>
</AccordionGroup>
