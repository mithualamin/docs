## Welcome to HTFlow

HTFlow is a comprehensive **workflow for creating page builder-supported HTML, CSS, and JavaScript with AI** and seamlessly converting them into **Webflow** or **Droip** (WordPress). 

Whether you're using AI tools like ChatGPT, Claude, Gemini, Cursor, or VS Code, HTFlow ensures your code follows page builder standards and converts accurately to production-ready sites.

## The HTFlow Workflow

HTFlow provides a complete ecosystem of tools to streamline your AI-powered development:

<Steps>
  <Step title="Build with AI">
    Use any AI model to generate HTML, CSS, and JavaScript that follows HTFlow rules for page builder compatibility
  </Step>
  
  <Step title="Develop Locally">
    Use HTFlow CLI and extensions in VS Code or Cursor to validate, test, and refine your code
  </Step>
  
  <Step title="Convert & Deploy">
    Transform your code into native Webflow or Droip using HTFlow's conversion tools
  </Step>
</Steps>

## How It Works

### Option 1: Standalone AI Development

<Steps>
  <Step title="Learn HTFlow Rules">
    Understand the HTML, CSS, and JavaScript standards for page builders
  </Step>
  
  <Step title="Use Any AI">
    Use ChatGPT, Claude, Gemini, or any AI model to generate code following HTFlow rules
  </Step>
  
  <Step title="Convert to Page Builder">
    Use HTFlow conversion tools to deploy to Webflow or Droip
  </Step>
</Steps>

### Option 2: Complete HTFlow Workflow (Recommended)

<Steps>
  <Step title="Set Up HTFlow Tools">
    Install HTFlow CLI, VS Code/Cursor extension, and Chrome extension
  </Step>
  
  <Step title="Build with AI">
    Use Cursor AI or VS Code with AI assistants to generate code that automatically follows HTFlow standards
  </Step>
  
  <Step title="Validate & Test">
    HTFlow CLI validates your code in real-time, ensuring page builder compatibility
  </Step>
  
  <Step title="Convert Accurately">
    Use HTFlow Webflow App or Chrome Extension to convert your project to native Webflow or Droip
  </Step>
</Steps>
