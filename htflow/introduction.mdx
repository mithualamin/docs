## Welcome to HTFlow

HTFlow is a development framework that creates **Webflow-compatible HTML, CSS, and JavaScript**. The HTFlow VS Code extension provides a visual interface to all HTFlow CLI commands, making it easy to manage projects without terminal commands.

## What is HTFlow?

HTFlow is a development framework designed for developers who want to create Webflow-compatible websites without learning complex terminal commands. With HTFlow, you can:

<AccordionGroup>
  <Accordion title="Initialize HTFlow Projects" icon="folder-plus">
    Quickly set up new HTFlow projects with the proper structure and configuration files.
  </Accordion>
  <Accordion title="Validate Code Compliance" icon="shield-check">
    Ensure your HTML, CSS, and JavaScript meet Webflow compatibility standards.
  </Accordion>
  <Accordion title="Run Project Audits" icon="clipboard-check">
    Get comprehensive analysis of your project with actionable insights and recommendations.
  </Accordion>
  <Accordion title="Start Development Servers" icon="code">
    Launch local development servers with hot reloading and automatic port management.
  </Accordion>
  <Accordion title="Build Production Code" icon="hammer">
    Create optimized, production-ready builds for deployment to Webflow.
  </Accordion>
</AccordionGroup>
