### CSS Optimization

```css
/* Use efficient selectors */
.button {
  /* ✅ Good - single class */
}
.card__title {
  /* ✅ Good - BEM methodology */
}

/* Avoid inefficient selectors */
div.container .button {
  /* ❌ Avoid - too specific */
}
* {
  /* ❌ Avoid - universal selector */
}

/* Use CSS containment for performance */
.card {
  contain: layout style paint;
}

/* Optimize animations */
.smooth-transition {
  will-change: transform;
  transform: translateZ(0); /* Force hardware acceleration */
}
```

### JavaScript Optimization

```javascript
// Use efficient DOM queries
const elements = document.querySelectorAll(".button");
const singleElement = document.querySelector(".hero-section");

// Cache DOM elements
class Component {
  constructor(selector) {
    this.element = document.querySelector(selector);
    this.buttons = this.element?.querySelectorAll(".button") || [];
    this.isActive = false;
  }

  // Use requestAnimationFrame for smooth animations
  animate() {
    requestAnimationFrame(() => {
      this.element.style.transform = "translateY(0)";
    });
  }
}

// Lazy load non-critical JavaScript
const loadNonCriticalJS = () => {
  const script = document.createElement("script");
  script.src = "/js/analytics.js";
  script.async = true;
  document.head.appendChild(script);
};

// Load after page is interactive
if (document.readyState === "interactive") {
  loadNonCriticalJS();
} else {
  document.addEventListener("DOMContentLoaded", loadNonCriticalJS);
}
```

```javascript
// Focus management
class FocusManager {
  constructor() {
    this.focusableElements =
      'button, [href], input, select, textarea, [tabindex]:not([tabindex="-1"])';
  }

  trapFocus(element) {
    const focusableElements = element.querySelectorAll(this.focusableElements);
    const firstElement = focusableElements[0];
    const lastElement = focusableElements[focusableElements.length - 1];

    element.addEventListener("keydown", (event) => {
      if (event.key === "Tab") {
        if (event.shiftKey) {
          if (document.activeElement === firstElement) {
            lastElement.focus();
            event.preventDefault();
          }
        } else {
          if (document.activeElement === lastElement) {
            firstElement.focus();
            event.preventDefault();
          }
        }
      }
    });
  }
}
```

## Common Pitfalls and Solutions

### Performance Issues

<AccordionGroup>
  <Accordion title="Large Bundle Sizes" icon="package">
    **Problem:** JavaScript bundle is too large **Solutions:** - Use dynamic
    imports for non-critical code - Implement code splitting - Remove unused CSS
    and JavaScript - Optimize images and use modern formats (WebP, AVIF)
  </Accordion>

  <Accordion title="Slow Loading" icon="clock">
    **Problem:** Pages load slowly **Solutions:** - Implement lazy loading for
    images - Use critical CSS inlining - Minimize render-blocking resources -
    Enable compression (gzip/brotli)
  </Accordion>

  <Accordion title="Layout Shifts" icon="move">
    **Problem:** Content jumps during loading **Solutions:** - Set explicit
    dimensions for images - Use CSS aspect-ratio for containers - Reserve space
    for dynamic content - Implement skeleton screens
  </Accordion>
</AccordionGroup>

### Conversion Issues

<AccordionGroup>
  <Accordion title="Styling Problems" icon="palette">
    **Problem:** Styles don't convert properly **Solutions:** - Ensure all
    styles use CSS classes - Avoid complex CSS selectors - Use standard CSS
    properties - Test with HTFlow audit before conversion
  </Accordion>

  <Accordion title="JavaScript Issues" icon="code-branch">
    **Problem:** JavaScript doesn't work after conversion **Solutions:** - Use
    class-based selectors only - Avoid ID-based JavaScript - Test all
    interactive features - Provide fallbacks for critical functionality
  </Accordion>
</AccordionGroup>
